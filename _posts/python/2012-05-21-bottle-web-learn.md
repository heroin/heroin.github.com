##安装bottle

访问bottle官网, [http://bottlepy.org](http://bottlepy.org/)

先在github上clone源码到本地进行安装

    git clone https://github.com/defnull/bottle.git
    cd bottle
    python setup.py install

安装完后打开python终端测试是否安装成功

    >>> import bottle
